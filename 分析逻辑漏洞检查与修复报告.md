## 二、新增：分析逻辑自检机制（任务1.3）

新增 [src/analyzer/self_check.py](src/analyzer/self_check.py)，每次生成报告前自动运行，预警写入报告固定章节（日报第九节/个股第八节），绝不静默：

| 自检维度 | 校验内容 |
|---|---|
| 规则参数一致性 | 报告引用的仓位上限必须等于 config 当前值×能力圈系数（防配置改后代码未同步） |
| 结论自洽性 | 有FAIL必须"排除/不满足"；无FAIL无MISSING不得判排除；评分<3必须标"仅观察"（裁决#6） |
| 数值合法性 | 评分∈[0,6]且=明细加和；市场状态∈config枚举；A/C区命中数=条件表判True数；情绪分∈[0,10] |
| 边界口径落实 | 亏损股PE口径、新股分位降级必须在判定明细中留痕 |

自检本身异常时输出P0级预警（不阻断报告生成）。防篡改回归：人为篡改总分/结论后自检均能捕获（test_tampered_score_detected / test_tampered_verdict_detected）。
