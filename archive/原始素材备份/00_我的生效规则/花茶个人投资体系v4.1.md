# 花茶个人投资体系 v4.1

> 建立日期：2026-04-07（v1.0）  
> 本版修订：2026-07  
> 核心理念：安全边际优先，纪律执行为王，认知复利驱动进化  
> 架构原则：**分层清晰、各司其职、上层约束下层、下层不反向干扰上层**

---

## 写在前面：投资自画像

我是一个偏结构性思维的个人投资者，资金规模在300万以下，不具备信息优势，也不具备专业机构的投研资源。我的优势是对行业趋势和企业基本面有较好的判断力，愿意持有，能扛回撤。我的致命弱点是仓位管理无纪律，导致过去三次大亏（重庆啤酒、招商轮船、中际旭创）都是"判断对了但执行错了"或"判断错了也没及时止损"。

这套体系的核心使命：**让正确的判断兑现为正确的结果，让错误的判断被及时纠正而不致命。**

### 体系设计原则

1. **四层递进**：战略配置→研究选择→交易执行→复盘进化，每层有明确的功能边界和执行频率
2. **上约束下**：上层决定下层的边界，下层不反过来干扰上层——第三层发现个股短期跌了10%，不回到第一层调整战略配置，除非触发了决策卡片上的止损条件
3. **两组合独立**：股息组合与择时组合物理隔离，在每一层内部分别定义规则，互不干扰
4. **规则嵌入防御**：人性弱点的防御不靠意志力，靠嵌入每个操作环节的硬性约束

### 收益来源信念

**股息组合**赚的是企业现金流分配的钱——找到持续分红、现金流稳定的优质企业，以低估值买入，长期持有，赚股息+估值修复的双驱动复利。

**择时组合**赚的是企业阶段性增长或估值回归的钱——分为成长线（赚龙头企业盈利增长）和修复线（赚被低估企业估值回归），持有周期1-6个月为主。

### 我不赚的钱

- 短期博弈、追涨杀跌、纯技术面炒作的钱
- 内幕消息、朋友推荐、小道消息的钱
- 自己看不懂的行业和商业模式的钱
- 情绪亢奋时追高的钱

## 第一层：战略配置层

> 回答"钱怎么分"。执行频率：每半年检视一次，或重大环境变化时触发调整。
> 日常操作不碰这一层——只在半年检视或系统性风险时调整。

### 1.1 资金分配与账户隔离

| 项目 | 股息组合 | 择时组合 |
|------|----------|----------|
| 资金比例 | 50% | 50% |
| 账户 | 独立券商账户A | 独立券商账户B |
| 市场 | A股 | A股 |
| 风格 | 长期持有，收息复利 | 中期波段，催化/轮动驱动 |
| 典型持有周期 | ≥1年 | 1-6个月，最长12个月 |

**防火墙规则（硬性，无例外）：**
- 两账户资金不互相调拨，任何理由都不行
- 分开记录、分开复盘、分开计算收益
- 股息组合的分红仅在本组合内再投资
- 择时组合的盈利仅在本组合内使用
- 一个组合的亏损不影响另一个组合的操作节奏

### 1.2 各组合核心指标

| 指标 | 股息组合 | 择时组合 |
|------|----------|----------|
| 年化收益目标 | 8%-12% | 20% |
| 最大回撤容忍 | ≤15% | ≤20% |
| 持股数量 | 5-10只 | ≤8只（实际持仓） |
| 单票仓位上限 | 15% | 25% |
| 单行业上限 | 30% | 40% |
| 现金最低留存 | 无硬性要求 | 非A区时≥10% |

### 1.3 市场状态判断（两组合共用）

| 状态 | 判定条件（满足≥3条进入该区） |
|------|------|
| **A区 · 低估** | ① 上证跌破3000点或市净率历史5%分位以下 ② Wind全A非金融石化PE＜22倍 ③ 股债利差＞5% ④ 全市场破净率＞10% ⑤ 日成交额连续10日＜7000亿且换手率＜1.5% ⑥ 官方明确"提振资本市场"表述 |
| **B偏低** | 股债利差4.5%-5%，成交额偏低，情绪偏冷 |
| **B中性** | 股债利差3.5%-4.5%，成交额正常 |
| **B偏高** | 股债利差3%-3.5%，成交额偏高，情绪偏热 |
| **C区 · 高估** | ① 上证逼近或突破历史高点 ② 恐惧贪婪指数极度贪婪，日成交持续＞1.5万亿 ③ 新股民开户数同比历史高位 ④ 偏股基金出现日光基 ⑤ 官方提示金融市场风险 ⑥ 股债利差＜2% |

**检查频率：** 每周日晚更新，记录在案，一周内不改判。

### 1.4 市场状态 → 仓位上限映射

| 市场状态 | 股息组合仓位上限 | 择时组合仓位上限 |
|----------|------------------|------------------|
| A区 | 100% | 100% |
| B偏低 | 80% | 70% |
| B中性 | 70% | 50% |
| B偏高 | 60%（暂停买入） | 40%（不开新仓） |
| C区 | 考虑减持高估标的 | 只卖不买，降至20%以下 |

### 1.5 择时组合严格风控模式

触发条件（任一）→ 仓位上限降至30%、单票上限降至15%、不开新仓：
- 账户年度累计盈利＞15%
- 市场进入C区
- 沪深300波动率（VIX类指标）＞35%

## 第二层：研究与选择层

> 回答"买什么"。执行频率：持续进行，但只在有明确机会时输出决策卡片。
> 这一层是投入认知精力最多的地方。

### 2.1 能力圈声明

| 能力圈等级 | 行业 | 操作约束 |
|------------|------|----------|
| **核心能力圈**（可重仓） | 新能源车、保险、储能、消费电子 | 单票可到对应组合上限 |
| **卫星能力圈**（可参与） | AI应用层、医疗器械、银行、煤炭、运营商 | 仓位减半约束 |
| **能力圈外**（不碰） | 创新药、农业、军工、看不懂的行业 | 禁止建仓 |

**完全排除（红线）：** ST股、退市风险股、纯概念股、财务数据有疑点的公司、大股东频繁减持/质押比例过高、审计意见异常、重大诉讼未决。

能力圈每年度复盘时可调整，但必须基于新增的系统性研究，不基于"最近赚了几次钱"。

### 2.2 股息组合选股标准

不同行业的财务特征差异巨大，用同一把尺子量会把优质标的误杀。选股标准分为**通用门槛**（所有行业共用）和**分类财务门槛**（按行业特性分别设定）。

#### 通用门槛（全部满足才进入候选池）

| 维度 | 指标 | 标准 |
|------|------|------|
| 流动性 | 市值 | ≥300亿 |
| 流动性 | 近20日日均成交额 | ＞1亿 |
| 合规 | 审计意见 | 标准无保留 |
| 合规 | 大股东质押率 | ＜30% |
| 合规 | 近3年监管函 | 无涉及财务真实性的 |
| 股息 | 连续分红 | ≥5年不间断 |
| 股息 | 派现比例（近3年均值） | ≥30% |
| 股息 | 当期股息率 | ≥1.5倍10年国债收益率，且绝对值≥3% |
| 股息 | 派息率 | ＜80%（分红留有余力） |

> 股息率设绝对底线3%，防止国债收益率极端下行时（如降至1.5%以下）门槛失效。

#### 分类财务门槛

**A类 · 金融（银行、保险）**
> 适用标的示例：招商银行、工商银行、建设银行、中国平安

| 指标 | 标准 | 说明 |
|------|------|------|
| ROE（近3年均值） | ≥8% | 行业净息差系统性下行，10%已是头部水平 |
| 不良贷款率 | ≤1.5% | 资产质量底线 |
| 拨备覆盖率 | ≥200% | 风险抵补能力 |
| 资本充足率 | ≥监管要求 | 经营安全性 |
| 净利润 | 近3年均为正 | 基本盈利能力 |
| 分红趋势 | 近3年不间断（允许金额波动） | 银行利润波动受拨备影响 |

不适用条件：收现比、净现比、资产负债率（金融行业这三项无参考价值）。

**B类 · 周期资源（煤炭、石化、有色）**
> 适用标的示例：中国神华、陕西煤业、中国石化

| 指标 | 标准 | 说明 |
|------|------|------|
| ROE（近3年均值） | ≥10% | 周期高点ROE极高，均值仍应保持合理水平 |
| 净利润 | 近3年中至少2年为正 | 允许周期底部偶发性下滑 |
| 收现比 | ≥80% | 周期底部收现略低可接受 |
| 净现比 | ≥70% | 资本开支较大，略放宽 |
| 资产负债率 | ≤65% | 周期行业负债率略高可接受 |
| 分红趋势 | 近5年不间断（允许金额随周期波动） | 核心看分红意愿的连续性，不苛求金额递增 |

**C类 · 稳定消费/公用事业/制造（消费、家电、水电、交运、运营商）**
> 适用标的示例：伊利股份、美的集团、长江电力、大秦铁路、中国移动

| 指标 | 标准 | 说明 |
|------|------|------|
| ROE扣非（近3年均值） | ≥12% | 稳定行业应有更高盈利质量要求 |
| 净利润 | 近3年均为正，同比降幅＜20% | 稳定性是核心诉求 |
| 收现比 | ≥90% | 收入质量高 |
| 净现比 | ≥80% | 利润含金量高 |
| 资产负债率 | ≤60% | 财务稳健 |
| 分红趋势 | 近3年不间断且非递减 | 稳定行业有能力做到 |

#### 加分条件（满足≥2条优先入选）

- 行业成熟期，需求稳定
- 行业长期增速≥3%
- 龙头/准龙头（市占率行业前三）
- 具备牌照/品牌/规模/成本壁垒
- 技术颠覆风险低

#### 估值买入标准（满足≥1条）

- PE ≤ 行业均值且处于自身历史30%分位以下
- PB ≤ 2.5 且处于自身历史30%分位以下（周期股优先看PB）
- 当期股息率 ≥ 2倍10年期国债收益率

### 2.3 择时组合备选池管理

**入池标准（全部满足）：**

| 维度 | 标准 |
|------|------|
| 理解度 | 能说清商业模式、竞争优势、主要风险、估值锚 |
| 市值 | ≥200亿 |
| 日均成交额 | 近5日＞1亿 |
| ROE（近3年均值） | ≥8% |
| 净利润 | 近3年中至少2年为正（允许困境反转标的因非经常性/周期性原因偶发亏损） |
| 大股东质押率 | ＜30% |
| 审计/监管 | 无重大问题 |

**池维护规则：**
- 总量30-50只，覆盖≥5个行业
- 每只标的必须有一份研究笔记（见附录A）
- 每季度审核一次，不达标移出，新标的经研究后补入

### 2.4 择时组合选股标准

分为**门槛条件**（必须全部满足）和**评分条件**（用于排序和仓位决策）。

**门槛条件（全部满足才可买入）：**

| 维度 | 条件 |
|------|------|
| 池内 | 标的必须在备选池内 |
| 估值 | PE分位≤历史30%且PE≤30，或PB≤2.5，或股息率≥4%（至少1条） |
| 位置 | 较1年内高点回撤≥25% |
| 基本面 | 最近一期财报营收或净利润同比不为负，或有明确业绩拐点预期 |
| 市场 | 非C区 |

**评分条件（每条1分，决定优先级和仓位大小）：**

| 评分项 | 说明 |
|--------|------|
| 技术底部 | 同花顺自编指标≥2个共振出现底部信号 |
| 右侧确认 | 周线MACD/RSI底背离，或行业指数站上60日线≥3日 |
| 政策催化 | 所属行业有国务院/部委级别明确利好 |
| 资金信号 | 北向资金连续2周净增持，或融资余额近期上升 |
| 机构预期 | 一致预期未来2年净利润CAGR≥15%，上调多于下调 |
| 预期差 | 存在市场未充分定价的正向因素（需在决策卡片中写明） |

**评分→仓位映射：**
- 5-6分：满额仓位（目标的100%）
- 3-4分：半额仓位（目标的50%-70%）
- 1-2分：轻仓试探（目标的30%以下）
- 0分：不买

### 2.5 择时组合ETF买入规则

ETF适用于个股不确定性高但行业方向明确的场景——无法判断哪家公司赢，但确信这个行业会涨。ETF不走个股筛选标准，单独设定规则。

**ETF入选条件（全部满足）：**

| 维度 | 标准 |
|------|------|
| 行业归属 | 跟踪行业属于能力圈范围内（核心或卫星） |
| 行业逻辑 | 有明确的行业级催化逻辑（政策/产业趋势/周期反转） |
| ETF规模 | ≥10亿元 |
| 日均成交额 | 近5日均值≥5000万 |
| 跟踪误差 | 近1年跟踪误差≤2% |

**ETF仓位约束：**
- 单只ETF占择时组合上限15%
- 全部ETF合计占择时组合上限30%
- ETF仍需填写决策卡片（核心假设改为行业级假设，止盈止损按择时规则执行）

### 2.6 决策卡片（统一入仓接口）

**任何标的从"想买"到"真买"，必须经过决策卡片。没有定稿的卡片，不准买入。**

决策卡片是这套体系中最重要的单一文档——它迫使每一个模糊的"感觉这个方向不错"转化成可证伪的具体假设。

## 第三层：交易执行层

> 回答"什么时候动手、怎么动手"。执行频率：由决策卡片和预设规则触发。
> 核心原则：所有操作必须是**预设规则触发的主动行为**，不是情绪驱动的被动反应。

### 3.1 股息组合交易规则

**建仓（分3批，间隔≥2周）：**

| 批次 | 占目标仓位 | 条件 |
|------|------------|------|
| 第1批 | 30% | 满足选股+估值标准 |
| 第2批 | 30% | 第1批后股价未涨超10%，或下跌≥5% |
| 第3批 | 40% | 股价较第1批成本持平或更低 |

**持有：** 核心持有≥1年，分红到账后再投入组合内股息率最高且估值最低的标的。红利税提醒：持有≥1年免征。

**卖出——强制（任一触发→1个月内清仓）：**
- 连续2年ROE＜8%
- 分红中断或派现比例连续2年＜25%
- 行业需求长期下滑（增速连续2年＜1%）
- 公司竞争优势明确丧失
- 财务造假、重大违规

**卖出——估值触发：**
- 股息率跌破10年国债收益率 → 分批减持，2个月内清仓
- PE/PB回升至历史70%分位以上 → 减持50%，观察1个月

**卖出——替换（每季度最多换1只）：**
- 发现更优标的（股息率高30%以上 + 估值更低 + 同等财务质量）

**再平衡：** 每半年一次（6月/12月底），单票＞15%减持，＜5%考虑加仓或清退；任一标的因涨幅＞20%立即减至15%。

### 3.2 择时组合交易规则

**建仓（分3批）：**

| 批次 | 占目标仓位 | 条件 |
|------|------------|------|
| 首仓 | 30% | 满足门槛+评分≥3 + 24小时冷静期 |
| 二仓 | 30% | 下跌≥5%，或横盘≥2周后放量突破 |
| 三仓 | 40% | 下跌≥8%，且基本面逻辑未变 |

**止盈（分档执行）：**

| 条件 | 操作 |
|------|------|
| 盈利达20%或达目标价 | 减持1/3，剩余用移动止盈 |
| 盈利达40%或PE升至历史70%分位 | 再减1/3 |
| 卖出信号（技术指标+放量滞涨） | 清仓剩余 |
| 自高点回落≥8% | 无条件清仓 |

**止损（硬性）：**

| 条件 | 操作 |
|------|------|
| 亏损达成本的-10% | 减仓50% |
| 亏损达成本的-15% | 清仓（无条件） |
| 买入逻辑被证伪 | 立即清仓，不看价格 |
| 突发黑天鹅 | 复牌首日评估，原则上清仓 |

**时间止损：**
- 3个月内既未盈利也未止损 → 减仓50%
- 6个月仍无起色 → 清仓
- 持有满12个月 → 必须重新评估，有新买入理由则继续，否则退出

### 3.3 通用硬性约束（两组合共用）

| 约束 | 规则 | 防御的弱点 |
|------|------|------------|
| 禁止高于均价加仓 | 股价高于持仓均价时不加仓 | 越涨越买 |
| 操作间隔 | 同一标的两次操作间隔≥5个交易日（股息组合≥2周） | 频繁交易 |
| 单日限制 | 每天最多建仓1只新标的 | 冲动决策 |
| 操作频率上限 | 择时每周最多2笔交易，股息每月最多1笔 | 过度交易 |
| 信息来源 | 任何建仓/清仓决策需≥2个独立信息源 | 单一信息依赖 |
| 情绪过滤 | 情绪评分1分或5分时，不执行非预案操作 | 恐惧/贪婪劫持 |
| 绝不借钱 | 配资、消费贷入市一律禁止 | 杠杆风险 |

### 3.4 情绪防御机制

**机制一：24小时冷静期**
- 适用：任何非预案内的临时操作
- 规则：记录想法 → 等24小时 → 次日收盘后决定
- 例外：触发硬性止损线立即执行

**机制二：连续亏损熔断**
- 连续3笔交易亏损 → 强制暂停开新仓2周，只持有或减仓

**机制三：情绪标记与回溯**
- 每次交易记录情绪分（1-5分）
- 月度复盘统计：极端情绪交易的亏损率 vs 平静时的亏损率
- 如果极端情绪交易亏损率＞60% → 下月冷静期升级为48小时

### 3.5 异常预案

| 场景 | 规则 |
|------|------|
| 单日账户亏损＞5% | 当日不做任何操作，次日收盘后评估 |
| 大盘单日跌幅＞5% | 不恐慌卖出；A区视为加仓机会 |
| 持仓股停牌 | 不调整其他仓位"补偿"，等复牌评估 |
| 政策突然利空 | 评估是否影响持仓逻辑，是→逻辑止损，否→不动 |
| 持仓股发布重大公告 | 公告日不操作，24小时消化后对照决策卡片判断 |
| 账户出现重大盈利 | 不加仓、不扩能力圈，按原计划执行 |
| 账户出现重大亏损 | 不摊薄、不换股、不借钱，启动冷静期 |

### 3.6 空仓期管理（择时组合专用）

- 空仓资金去向：国债逆回购或货币基金
- 空仓期行为：维护备选池、更新研究笔记、阅读研报，但不操作
- 认知锚定：**空仓是一种仓位决策，不是"什么都没做"**
- 对抗踏空焦虑：记录"想买但忍住了"的标的，事后复盘验证判断质量

## 第四层：复盘与进化层

> 回答"怎么持续提升判断力"。执行频率：按下表日历化执行。
> 核心原则：区分"判断"和"纪律"——现阶段守住纪律的优先级高于追求判断完美。

### 4.1 检查频率日历

| 事项 | 频率 | 耗时 | 内容 |
|------|------|------|------|
| 择时持仓检查 | 每日收盘后 | ＜5分钟 | 看日线，趋势在就不动（盘中不看） |
| 市场状态更新 | 每周日 | 15分钟 | 更新ABC区判断 |
| 股息持仓检查 | 每周一次 | 10分钟 | 看周线，有无异常 |
| 股息标的基本面 | 每月一次 | 30分钟 | 财务数据、分红公告 |
| 月度复盘 | 每月最后交易日 | 30分钟 | 见下方 |
| 备选池维护 | 每季度一次 | 2小时 | 增删标的、更新研究笔记 |
| 季度复盘 | 每季度末 | 1-2小时 | 见下方 |
| 股息组合再平衡 | 6月/12月底 | 30分钟 | 检查仓位偏离 |
| 战略配置检视 | 每半年 | 1小时 | 第一层参数是否需要调整 |
| 年度复盘 | 12月 | 半天 | 见下方 |

### 4.2 复盘机制

**月度复盘（30分钟，每月最后交易日）：** 回答5个问题并记录

1. 本月有没有违反纪律？具体哪条？
2. 盈亏最大的一笔交易，决策过程有无问题？
3. 情绪评分统计：极端情绪交易占比？
4. 市场状态是否需要更新？
5. 下月需要调整什么？

**季度复盘（1-2小时）：**
- 两组合分别计算季度收益率、最大回撤
- 择时组合：胜率和盈亏比是否达标
- 股息组合：持仓标的基本面变化、分红是否正常
- 备选池季度更新
- 判断×纪律象限归因（对了赚钱/对了少赚/错了小亏/错了大亏）

**年度复盘（半天）：**
- 全年收益率，与沪深300对标
- 纪律违反统计：哪条最频繁？是规则设计问题还是执行力问题？
- 能力圈复核：圈内操作胜率是否更高？有无升级/降级候选？
- 体系迭代：每年最多修改2-3条规则
- 所有参数阈值审视

### 4.3 体系迭代规则

- 参数调整基于≥6个月数据回顾，不因单次交易修改
- 每次修改记录：修改原因、前后参数、生效日期
- 同一参数一年内最多修改2次
- 重大迭代需月度复盘中提出，冷静一个月后执行
- 修订必须在非交易时段、非情绪化状态下进行
- 所有迭代决策写入《体系演化日志》

### 4.4 认知沉淀机制

**认知卡片：** 每当形成一个新的认知框架或总结出一条可复用的投资规律，浓缩成一页纸的"认知卡片"存入知识库。不是为了重复使用同一框架，而是类似场景再现时能快速调取。

**研究笔记：** 备选池内每只标的配一份简要研究笔记（见附录A），跟随季度备选池维护一起更新。

**外部观点加工：** 所有外部研报、文章、观点，按《外部观点加工协议》处理，转化为体系迭代的潜在素材。核心产出是：这个观点对我的体系是印证、冲突、还是补充？

**核心习惯：** 任何外部观点必须经加工后才能进入决策流程——不允许"刚看完就买"。

## 附录B 核心教训（置顶提醒）

> 1. **当真正该买的时候，你一点都不想买** → 所以在情绪平静时预设决策卡片，满足条件就执行首仓
> 2. **越涨越买，浮盈亏光** → 所以禁止在高于均价时加仓
> 3. **频繁交易，没按计划加仓** → 所以设操作频率上限和最小操作间隔
> 4. **趋势交易盯盘是自我伤害** → 所以规定检查频率，盘中不看
> 5. **充分研究的投资，可以卖一部分但保留一些** → 所以分档止盈
> 6. **对的判断+错的仓位管理=亏钱** → 所以仓位管理和判断同等重要

## 附录D 四层全景

```
第一层 战略配置 ──── 半年一动，定框架
  "钱怎么分？市场在哪个区？"
  │
  ▼ 资金分配到两个账户，市场状态约束仓位上限
  │

  "买什么？为什么买？"
  │
  ▼ 决策卡片指导具体操作
  │
第三层 交易执行 ──── 规则触发，按纪律
  "什么时候动手？怎么止盈止损？"
  │
  ▼ 操作结果反馈到复盘
  │
第四层 复盘进化 ──── 按节奏，沉淀认知
  "哪里做对了？哪里做错了？体系怎么改？"
  │
  └──────▶ 反哺第一、二层的判断质量
```

**核心约束：上层决定下层边界，下层不反过来干扰上层。**
